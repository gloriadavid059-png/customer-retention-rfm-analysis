# Customer Behavior Analysis (RFM Project)

## 📊 Project Overview
This project analyzes customer behavior using RFM (Recency, Frequency, Monetary) analysis to segment customers and evaluate revenue contribution patterns.

The goal was to identify high-value customers, understand purchasing behavior, and uncover retention opportunities using data-driven insights.

---

## 🛠 Tools Used
- PostgreSQL for data extraction and analysis  
- Excel for data validation and calculations  
- Tableau for dashboard visualization  

---

## 📂 Project Workflow

### 1. Data Exploration
Explored customer, order, and order item datasets to understand structure and relationships.

### 2. Data Cleaning & Joining
Performed SQL joins between:
- Customers table  
- Orders table  
- Order items table  

### 3. Customer Analysis
Calculated:
- Recency (how recently a customer purchased)
- Frequency (number of purchases)
- Monetary value (total spending)

### 4. RFM Analysis
Segmented customers based on RFM metrics to identify:
- VIP Customers  
- High Value Customers  
- Regular Customers  
- Lost Customers  

### 5. Dashboard Creation
Built an interactive dashboard in Tableau to visualize customer segmentation and revenue distribution.

---

## 📊 Key Insights
- VIP customers contribute the highest revenue  
- Majority of customers fall into low-value segments  
- Revenue is highly concentrated among a small group of customers  
- Strong opportunity exists for customer retention strategies  

---

## 📌 Project Outcome
This analysis helps businesses:
- Improve customer retention  
- Identify high-value customers  
- Optimize marketing strategies  
- Increase overall revenue efficiency
